# Canonical Terms

This file contains the fixed English labels to use in this project when translating or reviewing SGS rules text. The Chinese term is authoritative; the English term is the enforced project label.

- `视为` -> `is treated as`
- `使用` -> `Use`
- `打出` -> `Play`
- `失去体力` -> `Lose HP`
- `受到伤害` -> `Take damage`

- `取消` -> `Remove target`

- `无视` -> `Ignore`
- `展示` -> `Show`

## Usage Notes

- `视为` must stay stronger than `为`. Do not rewrite them as interchangeable English.
- `展示` and `亮出` must not share the same bare English verb in final copy, because the rules distinguish temporary showing from turning face-up.
- `无视` is not the same as `无效`: ignoring a condition or armor is different from nullifying a skill/effect.
- `代替` and `替换` describe different destination outcomes for the original card; preserve that difference explicitly.
